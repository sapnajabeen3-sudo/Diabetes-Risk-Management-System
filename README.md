# 🩺 Diabetes Risk Management System

## Overview

The Diabetes Risk Management System is a Machine Learning-based application developed to assess an individual's risk of diabetes using health-related parameters. The system analyzes user input and provides a prediction along with basic health recommendations to promote early awareness and preventive care.

## Features

- Diabetes risk prediction using Machine Learning
- Simple and user-friendly interface
- Real-time prediction results
- Health recommendations based on risk level
- Fast and efficient data processing
- Interactive and easy-to-understand output

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-Learn
- Plotly

## How It Works

1. The user enters health-related information.
2. The system processes the input data.
3. The trained Machine Learning model analyzes the data.
4. A diabetes risk prediction is generated.
5. The user receives the prediction result and relevant recommendations.

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Diabetes-Risk-Management-System.git
```

Move to the project directory:

```bash
cd Diabetes-Risk-Management-System
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
